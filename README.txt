Ejercicio Traslado Versión 2.0 28/12/2015

FUNDAMENTOS
-----------

- Se solicita migrar alumnos desde un txt con nº de alumnos desconocido, a la base de datos Iparsex.


NOTAS DE USO
------------

- El programa tiene una única clase ejecutable denominada "Traslado".

- El archivo de lectura debe estar dentro de una carpeta "data" que se encuentra en la raíz del programa. Ese archivo ha de llamarse "personas.txt".

- Dicha carpeta inluye el archivo "create.sql" con las instrucciones para crear la tabla "iparsex", con usuario "root" y sin contraseña.

- Finalizada la ejecución del programa, se crea un informe dentro de la carpeta "data".
